# Development Guide\n\nTo get started with the development environment:\n\n## Prerequisites\n\n- Node.js 18 or higher\n- PostgreSQL 12 or higher\n- npm or yarn\n\n## Setup\n\n1. Clone the repository\n2. Install dependencies: `npm install`\n3. Copy `.env.example` to `.env.local` and update values\n4. Create PostgreSQL database: `createdb app_db`\n5. Run migrations: `npm run db:generate && npm run db:migrate`\n6. Start dev server: `npm run dev`\n\n## Database\n\n- Uses Drizzle ORM for type-safe database queries\n- PostgreSQL for persistence\n- Auto-migrations with Drizzle Kit\n\n## Available Commands\n\n- `npm run dev` - Start development server\n- `npm run build` - Build for production\n- `npm start` - Start production server\n- `npm run lint` - Run ESLint\n- `npm run typecheck` - TypeScript type checking\n- `npm run db:generate` - Generate Drizzle schema\n- `npm run db:migrate` - Run database migrations\n\n## Project Structure\n\n```\nsrc/\n‚îú‚îÄ‚îÄ app/               # Next.js app directory\n‚îú‚îÄ‚îÄ db/                # Database configuration\n‚îú‚îÄ‚îÄ lib/               # Utility functions\n‚îú‚îÄ‚îÄ components/        # React components\n‚îî‚îÄ‚îÄ middleware.ts      # Next.js middleware\n```\n\n## Authentication\n\n- JWT-based authentication\n- Session cookies (httpOnly)\n- Role-based access control (RBAC)\n\n## Components\n\nUI components are located in `src/components/ui/` and include:\n- Button\n- Input\n- Card\n- CardHeader\n- CardTitle\n- CardContent\n\n## API Routes\n\nAll API routes are in `src/app/api/`\n\n## Styling\n\n- Tailwind CSS for styling\n- Responsive design\n- Dark mode ready\n\n## Troubleshooting\n\n### Database Connection Error\n\nEnsure PostgreSQL is running and connection string is correct in `.env.local`\n\n### Port Already in Use\n\nThe default port is 3000. Use `PORT=3001 npm run dev` to use a different port.\n\n### Build Errors\n\nClear cache and rebuild:\n```bash\nrm -rf .next\nnpm run build\n```\n
